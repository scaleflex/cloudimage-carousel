CloudImage Carousel

Two initialization methods — JavaScript API and HTML data-attributes

1. JavaScript API

const carousel = new CloudImageCarousel('#carousel-js', {
  images: ['hotel.jpg', 'birds.jpg', ...],
  theme: 'light',
  showBullets: true,
  transitionEffect: 'slide',
})
carousel.init()

2. HTML Data Attributes + autoInit()

<div
  data-ci-carousel-images='["hotel.jpg", "birds.jpg", ...]'
  data-ci-carousel-theme="dark"
  data-ci-carousel-show-bullets="true"
  data-ci-carousel-transition="fade"
></div>

<script>CloudImageCarousel.autoInit()</script>
